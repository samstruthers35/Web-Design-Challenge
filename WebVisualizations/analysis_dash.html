Summary: Latitude vs. X

The purpose of this project was to analyze the effect of location on weather. We wanted to know how the weather would change in relation to how close you were to the equator. To accoomplish this, we used the OpenWeatherMap API to assemble a dataset with over 500 cities.



After Assembling the dataset, we used MatPlotLib to plot all the datapoints. We looked at certain factors such as Tempereature, Wind Speeds, Humidity and Cloudiness. This site provides the data along with the vizuals that were created as part of the analysis.
